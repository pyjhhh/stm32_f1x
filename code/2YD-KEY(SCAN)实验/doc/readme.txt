使用编译环境：MDK4.72A  IAR6.30
使用硬件环境：STM32F103RBT6
使用函数库版本：STM32F10x_StdPeriph_Driver V3.50
功能概要:按下S1，S2，S3函数进入按键扫描函数,长时间按的话，相应的led灯会闪烁S1->D1;S2->D2;S3->D4;  

调试说明：此例程适用的调试下载器是J-link-ob，也可以用过串口下载调试程序,（请USB线插在U2上,打开串口下载程序软件，配置波特率115200，8位，一个停止位，无校验位。）
